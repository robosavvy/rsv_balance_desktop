# rsv_balance_desktop
Desktop packages for RoboSavvy's balancing platform

---
## Packages

* `rsv_balance_rqt`
  
  RQT plugin for selecting between operating modes.
  
* `rsv_balance_viz`

  Launch files for RVIZ URDF model and robot visualization.
